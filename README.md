# 📖 Projeto mdBook - Blog estático

Este projeto utiliza [mdBook](https://rust-lang.github.io/mdBook/) para gerar um site estático a partir de arquivos Markdown. Ele é configurado para publicar o site diretamente no branch `gh-pages` do GitHub, sem necessidade de GitHub Actions.

## 📦 Pré-requisitos

- Ubuntu (ou similar com `apt`)
- Acesso à internet para baixar os instaladores

## 🚀 Instalação automática

Execute o script `install.sh` para instalar todas as dependências necessárias:

```bash
chmod +x install.sh
./install.sh
